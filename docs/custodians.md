# Responsáveis patrimoniais

Responsáveis patrimoniais representam pessoas ou funções que recebem guarda, uso ou empréstimo de ativos. Eles são separados de usuários para preservar histórico mesmo quando uma conta é bloqueada ou removida.

Campos sensíveis, como documento identificador, não aparecem em listagens. Matrícula é única por tenant quando informada. Responsáveis em uso devem ser inativados, não apagados.

Todo responsável pertence a um tenant e a uma unidade administrativa. O vínculo com usuário é opcional e, quando informado, deve pertencer ao mesmo tenant.
